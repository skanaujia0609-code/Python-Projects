#Python Data Analysis Projects

This repository contains a collection of Python projects focused on automation, data cleaning, API integration, web scraping, and data visualization.
Each project is implemented using Python and Jupyter Notebook, demonstrating practical real-world use cases for data analysis.

#Projects Included
1)Amazon Web Scraping using Python
Scrapes product details from Amazon.
Uses requests, BeautifulSoup, and data extraction logic.
Demonstrates real-world web scraping workflow.

2)Automatic File Sorter in File Explorer
Automatically organizes files by type.
Uses os and shutil modules.
Helps clean and manage local storage efficiently.

3)Automating Crypto Website API Pull (Python)
Fetches live crypto data using API calls.
Uses requests and JSON parsing.
Automates cryptocurrency price tracking.

4)Data Cleaning using Pandas
Dataset Used: Customer Call List.xlsx
Removes duplicates, handles missing values, cleans phone numbers, and corrects text formatting.
Uses pandas to transform raw customer call data into a clean, usable format.
Demonstrates practical data preprocessing steps.

5)Pandas Visualization
Dataset Used: Ice Cream Ratings.csv
Creates visualizations like bar charts and line charts using matplotlib and pandas.
Shows ratings and comparisons across ice cream flavors.
A beginner-friendly visualization project.
